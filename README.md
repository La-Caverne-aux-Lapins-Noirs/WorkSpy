# Persoc

Multifunction software, made to be installed on Huile operating system in a Technocore/Infosphere managed computer lab.

* Contains a spyware that detects activities on the machine and users responsible for it (to report to Infosphere and say those users are currently working)
* A signal sending device that kill sessions that are not supposed to be open (for exam - if you are not concerned, you must no be in the rooms it occurs!)
